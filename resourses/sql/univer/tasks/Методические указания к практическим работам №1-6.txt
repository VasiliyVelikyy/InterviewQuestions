Методические указания к практическим работам №1-6
Практическая работа №1. Создание простых запросов с использованием операторов языка манипулирования данными (SELECT, UPDATE, INSERT, DELETE).

Цель работы: Научиться, используя язык манипулирования данными SQL, работать с данными в СУБД ORACLE. Приобрести навыки построения сложных запросов на выборку, обращающихся к нескольким таблицам с помощью соединения, объединения, декартового произведения, разности, пересечения, а также используя подзапросы.

Практическая работа №2. Процедуры и функции. Хранимые процедуры и функции. Разработка и использование хранимых процедур.

Цель работы: Научиться, используя диалект языка SQL - PL/SQL, разрабатывать процедуры и функции для обработки данных. Приобрести навыки работы с управляющими конструкциями и операторами языка PL/SQL.

Практическая работа №3. Обработка исключительных ситуаций.

Цель работы: Научиться, с помощью PL/SQL, обрабатывать различные исключительные ситуации, возникающие при вызове процедур, функций, пакетов, и выводить их на экран пользователя с помощью функции назначения текста ошибки.

Практическая работа №4. Создание триггеров для обеспечения ссылочной целостности.

Цель работы: Научиться создавать и работать с триггерами, которые срабатывают при наступлении какого-либо события, а также триггеры, замещающие действия в базе данных.
